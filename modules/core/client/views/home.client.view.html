Customer Manager

This app has been developed on MEANjs Framework.It helps the user to create,edit,remove and store customres.


  • Backend Technologies used

    • NODE.JS runs on the server side

  • Frontend Technologies used

    • Anular.js   manages the App on the client side
    • Anular-routes.js   manages the client routes
    • Anular-ui.js,Bootstrap   For user interface display

  • Features

    • Add articles   Any user (even without an account) can post articles
      which are then verified by the Admin
    • Global Chat  The chat feature uses socket.io to enable realtime chat exprience
    • Customers  Can be only accessed by registered Users It allowes the users to create,remove and edit a customer
    • Todo  Can be only accessed by registered Users It allowes the users to create a todo list
